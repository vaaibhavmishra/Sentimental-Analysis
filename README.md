# Sentiment Analysis Project

## How to Run

1. **Prerequisites**: Ensure you have Python 3.9+ installed.

2. **Setup and Run**:
   You can easily run the project using the provided script:
   ```bash
   chmod +x run_project.sh
   ./run_project.sh
   ```

   Or manually:
   ```bash
   # Create virtual environment
   python3 -m venv venv
   source venv/bin/activate

   # Install dependencies
   pip install -r requirements.txt

   # Run the app
   python app1.py
   ```

3. **Access**:
   Open your browser and navigate to `http://127.0.0.1:8000`.

**Note**: On the first run, the application will download a large model file (approx. 700MB). Please be patient.
